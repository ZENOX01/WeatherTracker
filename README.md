# 🌦 Weather App

A simple weather application built using **Vanilla JavaScript**, HTML, and CSS.

This app allows users to search for any city and view real-time weather data using the OpenWeather API.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌡 Toggle between Celsius (°C) and Fahrenheit (°F)
- ⌨ Press "Enter" to search
- ⚡ Loading state while fetching data
- ❌ Error handling for invalid cities or failed requests
- 🎨 Modern styled UI

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- OpenWeather API

## 🌍 How It Works

1. User enters a city name.
2. App sends a request to OpenWeather API.
3. Weather data is fetched and displayed.
4. User can toggle between °C and °F.
5. Errors are handled gracefully if city is not found.
